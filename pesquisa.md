Bootstrap é um framework front-end amplamente utilizado que reúne um conjunto organizado de CSS, JavaScript e componentes prontos, facilitando a criação de interfaces modernas. Ele se destaca por oferecer um sistema de grid responsivo, que adapta o layout a diferentes tamanhos de tela, além de uma variedade de elementos reutilizáveis, como botões, cards e navegações. Por ser baseado em padrões consistentes, o Bootstrap permite que desenvolvedores criem páginas de forma rápida e padronizada, reduzindo o tempo de escrita de código. Além disso, sua documentação extensa e comunidade ativa tornam o aprendizado acessível. Outro ponto relevante é a customização via Sass e variáveis CSS, que possibilitam ajustar cores, espaçamentos e estilos sem grandes esforços. Como resultado, o framework combina agilidade, organização e compatibilidade, sendo uma solução prática tanto para protótipos quanto para projetos profissionais.
Escolhemos o Bootstrap para o projeto pois ele é um framework fácil de utilizar e de aprender, permitindo que o desenvolvimento avance de forma rápida. Além disso, como oferece componentes prontos e bem estruturados, reduz o tempo gasto na criação de layouts e garante um visual organizado desde o início. E sua documentação clara ajuda a tirar dúvidas rapidamente, o que evita atrasos no processo. Por fim, por ser amplamente usado e testado, o Bootstrap também nos transmitiu segurança, praticidade e confiabilidade, tornando-se uma ótima opção para nosso projeto.
